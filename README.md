CITYPULSE — Simulador de tráfico urbano en una intersección controlada por semáforos.
Permite configurar tasas de llegada de vehículos, duración de fases semafóricas y eventos externos (accidentes, lluvia),
y observar en tiempo real métricas como tiempo de espera, longitud de cola y nivel de congestión. Desarrollado en Java aplicando principios de POO:
encapsulamiento, herencia y polimorfismo.
